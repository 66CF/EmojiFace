# Model Download Required

The application expects the base YOLOv8n-Face ONNX model at this location:
`web/public/models/yolov8n-face.onnx`

Please download it from:
https://github.com/derronqi/yolov8-face/raw/main/yolov8n-face.onnx

Or any compatible YOLOv8-face model that takes [1, 3, 640, 640] input and outputs [1, 20, 8400].
